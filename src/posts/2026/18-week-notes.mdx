## The Responder

I read [this review](https://www.theguardian.com/tv-and-radio/2026/apr/26/the-cage-review-bbc-one-iplayer-michael-socha) of _The Cage_ the other day and thought it sounded good. But the review mentioned the writers previous drama _The Responder_ so we’ve been watching that first. Only one more episode to go. I’ve really enjoyed it, I was dubious about Martin Freeman doing a scouse accent, but he does a great job! It’s quite bleak and intense but it’s a good story full of great characters. And it’s fun watching something set in Liverpool.

## Varlock

A while ago I [migrated](https://github.com/jonheslop/jfh-2025/pull/14) this site to use [Varlock](https://varlock.dev/). Before that I just kept secrets locally in an `.env` file and this feels very sophisticated for a personal project. I’ve stored the actual secrets in 1Password and when I run `bun dev` locally it feels very future that a little popup asks for my fingerprint to start things up.

I heard about it via bluesky but I think the thing that made me more keen was a few weeks prior I was showing how [gif.land](https://gif.land/) works to people at work and whilst sharing my screen I tabbed past the `.env` file and everyone took the piss saying they were gonna ruin me now they had my Cloudflare API key. Anyway it was funny and I felt silly, so when I saw [Varlock](https://varlock.dev/), I was like “oh I get it now!”
